# 春辉综合服务部 — 微信小程序

> 面向乡镇、县城下沉市场的本地综合生活服务平台小程序

## 项目介绍

春辉综合服务部是一款本地化O2O生活服务超级平台，整合了保洁清洗、养老陪护、搬家安装、家政月嫂、婚姻介绍、招聘求职等高频生活服务板块。平台支持普通用户、工人/师傅、合作商、管理员四方角色，通过统一的身份切换机制，各角色拥有完全独立的功能空间。

## 技术栈

- **前端**: 微信小程序原生框架（WXML + WXSS + JS）
- **后端**: 微信云开发（CloudBase）— 云函数 + 云数据库 + 云存储
- **支付**: 微信支付（JSAPI支付）
- **地图**: 腾讯位置服务
- **通讯**: 云开发实时数据推送

## 快速开始

### 环境要求
- 微信开发者工具 ≥ 1.06
- Node.js ≥ 16.x
- 微信小程序账号（需开通云开发）

### 安装步骤

1. 克隆项目
```bash
git clone <repository-url>
cd chunhui-miniprogram
```

2. 安装云函数依赖
```bash
cd cloud/functions
for dir in */; do
  cd "$dir" && npm install && cd ..
done
```

3. 导入微信开发者工具
   - 打开微信开发者工具
   - 选择「导入项目」
   - 填写你的 AppID
   - 后端服务选择「微信云开发」

4. 配置云开发
   - 点击「云开发」按钮开通
   - 在 config.js 中填入你的云开发环境ID
   - 部署云函数（右键 cloud/functions 目录 → 创建并部署所有云函数）
   - 初始化数据库集合

5. 运行项目
   - 点击「编译」即可预览

## 核心功能

### 四方角色体系
- **普通用户**: 浏览服务、预约下单、查看订单
- **工人/师傅**: 发布职业信息、接单抢单、收益管理
- **合作商**: 发布招聘/婚恋信息、内容管理
- **管理员**: 审核管理、数据统计、系统配置

### 核心业务流程
1. 用户浏览服务 → 选择工人 → 预约下单 → 工人接单 → 完成服务 → 评价
2. 工人入驻 → 提交资料 → 管理员审核 → 通过上线
3. 合作商发布信息 → 管理员审核 → 前端展示
