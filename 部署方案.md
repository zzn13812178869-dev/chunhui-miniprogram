# 春辉综合服务部小程序 — 部署方案

## 一、部署前置条件

| 条件 | 说明 | 状态 |
|------|------|------|
| 微信小程序账号 | 需企业/个体户认证 | 需用户准备 |
| 微信开发者工具 | 最新稳定版 | 需下载安装 |
| 微信云开发环境 | CloudBase | 需在小程序后台开通 |
| 腾讯位置服务Key | 用于地图选址 | 需申请 |

---

## 二、环境配置

### 2.1 修改配置文件

文件：`project/config.js`

```javascript
// 必须修改的配置项
cloudEnvId: 'your-cloud-env-id',        // 替换为你的云开发环境ID
tencentMapKey: 'YOUR-TENCENT-MAP-KEY',  // 替换为你的腾讯地图Key
servicePhone: '400-888-8888',           // 替换为实际客服电话
```

### 2.2 获取云开发环境ID

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入小程序后台 → 云开发 → 概览
3. 复制「环境ID」填入 `config.js`

---

## 三、数据库初始化

### 3.1 创建集合（10个）

在云开发控制台 → 数据库中创建以下集合：

| 集合名称 | 说明 | 必须索引 |
|---------|------|---------|
| `users` | 用户信息 | `_openid`（唯一） |
| `workers` | 工人职业信息 | `_openid`, `status` |
| `orders` | 订单 | `userId`, `workerId`, `status` |
| `reviews` | 评价 | `orderId`, `workerId` |
| `messages` | 聊天记录 | `chatId`, `receiverId` |
| `posts` | 发布内容 | `partnerId`, `status` |
| `banners` | 轮播图 | `status`, `sortOrder` |
| `withdrawals` | 提现记录 | `userId`, `status` |
| `income_records` | 收益记录 | `userId` |
| `news` | 资讯 | `category`, `isOnline` |

### 3.2 初始化数据

在 `banners` 集合中插入初始轮播图数据：

```json
{
  "image": "cloud://your-env-id/banners/banner1.jpg",
  "title": "专业搬家服务",
  "link": "",
  "sortOrder": 1,
  "status": "active"
}
```

## 四、云函数部署

### 4.1 上传云函数

在微信开发者工具中：

1. 右键 `cloud/functions/login` → 「创建并部署：云端安装依赖」
2. 依次对所有云函数执行上述操作：
   - login, registerUser, getUserInfo, updateUserInfo
   - getBannerList, getCategoryList, getNewsList, getNewsDetail
   - getWorkerList, getWorkerDetail, createOrder, cancelOrder, grabOrder
   - getOrderList, getOrderDetail, updateOrderStatus, submitReview
   - uploadImage, roleSwitch

### 4.2 云函数批量部署脚本

```bash
# 在微信开发者工具控制台中执行
const functions = [
  'login', 'registerUser', 'getUserInfo', 'updateUserInfo',
  'getBannerList', 'getCategoryList', 'getNewsList', 'getNewsDetail',
  'getWorkerList', 'getWorkerDetail', 'createOrder', 'cancelOrder', 'grabOrder',
  'getOrderList', 'getOrderDetail', 'updateOrderStatus', 'submitReview',
  'uploadImage', 'roleSwitch'
];

functions.forEach(name => {
  wx.cloud.callFunction({ name, data: {} }).catch(() => {});
});
```

## 五、小程序上传与发布

### 5.1 本地预览

1. 打开微信开发者工具
2. 导入项目（选择 `project/project.config.json`）
3. 点击「预览」生成二维码
4. 微信扫码预览

### 5.2 代码上传

1. 点击开发者工具右上角「上传」
2. 填写版本号（如 `1.0.0`）
3. 填写项目备注
4. 点击「上传」

### 5.3 提交审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入「版本管理」→「开发版本」
3. 点击「提交审核」
4. 填写小程序信息：
   - 服务类目：生活服务 > 家政服务
   - 标签：家政、保洁、维修、搬家
   - 测试账号（审核用）：提供管理员账号密码

### 5.4 审核通过后发布

审核通过后，在「版本管理」中点击「发布」即可上线。

## 六、支付配置（可选，M2阶段）

### 6.1 微信支付开通

1. 小程序后台 → 微信支付 → 开通
2. 绑定商户号（需企业资质）
3. 配置支付目录：`https://[云开发环境ID].service.tcloudbase.com/`

### 6.2 云函数支付配置

在 `cloud/functions/getUnifiedOrder/index.js` 中配置商户信息：

```javascript
const mchId = 'YOUR_MCH_ID';        // 商户号
const apiKey = 'YOUR_API_KEY';       // API密钥
```

## 七、运维监控

### 7.1 云开发监控

- 云函数调用次数/耗时
- 云数据库读写次数
- 云存储流量

### 7.2 告警设置

在云开发控制台设置用量告警：
- 云函数并发 > 800
- 数据库容量 > 80%
- 云存储容量 > 80%

### 7.3 日志查看

```javascript
// 在云函数中使用日志
console.log('操作日志', { userId, action, result });
```

在云开发控制台 → 日志中查看。
